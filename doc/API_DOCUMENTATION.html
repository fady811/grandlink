GrandLink API

Complete backend API reference for developers

Authentication

POST /api/auth/register/

Register a new user account. Creates an inactive user and sends an OTP to their email for verification.

Request Body 
{
  "email": "user@example.com",
  "password": "Password123!",
  "password2": "Password123!",
  "role": "student" 
}
Success Response (201) 
{
  "message": "User created. Please verify email.",
  "email": "user@example.com"
}
POST /api/auth/verify-otp/

Verify user email using the 6-digit code sent during registration or resend.

Request Body 
{
  "email": "user@example.com",
  "code": "123456"
}
Success Response (200) 
{
  "message": "Email verified successfully.",
  "access": "jwt_access_token",
  "refresh": "jwt_refresh_token"
}
POST /api/auth/login/

Authenticate user and obtain JWT tokens.

Request Body 
{
  "email": "user@example.com",
  "password": "Password123!"
}
Success Response (200) 
{
  "access": "jwt_token_here",
  "refresh": "jwt_refresh_here",
  "user": {
    "id": "uuid",
    "email": "user@example.com",
    "role": "student"
  }
}
POST /api/auth/google/

Authenticate via Google OAuth2 token. Automatically creates/activates user.

Request Body 
{
  "token": "google_id_token",
  "role": "employer"
}
Success Response (200) 
{
  "access": "jwt_token",
  "refresh": "jwt_refresh",
  "user": { ... }
}

Profiles

GET PUT /api/profiles/student/[UUID]/ JWT Required

Retrieve or update student profile details. Path parameter is optional for the owner.

Example PUT Body 
{
  "university": "Cairo University",
  "major": "Computer Science",
  "graduation_year": 2025,
  "gpa": "3.85",
  "skills": ["Python", "Django", "React"],
  "hide_gpa": false
}
Response Example 
{
  "id": "uuid",
  "university": "Cairo University",
  "major": "Computer Science",
  "graduation_year": 2025,
  "gpa": "3.85",
  "bio": "Passionate developer...",
  "skills": ["Python", "Django"],
  "is_profile_public": true
}
GET PUT /api/profiles/employer/[UUID]/ JWT Required

Manage employer profile. Requires authenticated user with employer role or admin access.

Example Body 
{
  "company_name": "TechCorp",
  "industry": "Software",
  "company_size": "50-100",
  "website": "https://techcorp.com",
  "phone": "+20123456789"
}
Response Body 
{
  "id": "uuid",
  "company_name": "TechCorp",
  "industry": "Software",
  "is_verified": true,
  "is_profile_public": true
}
PATCH /api/profiles/privacy/ JWT Required

Endpoint to quickly toggle privacy flags for the current user's profile.

Request Body 
{
  "hide_gpa": true,
  "hide_phone": true,
  "is_profile_public": false
}
Response Body 
{
  "message": "Privacy settings updated.",
  "status": "success"
}
